# ACC Skill Verification Case — Right Hard Palate ACC

## Case Summary

| Parameter | Detail |
|-----------|--------|
| Primary site | Right hard palate / soft palate junction |
| Histology | Adenoid cystic carcinoma, cribriform-tubular type |
| Surgery | Extended maxillectomy, full-thickness hard palate → nasal floor invasion |
| MRI | Ipsilateral V2 widening at pterygopalatine fossa / infraorbital canal |
| Symptoms | Ipsilateral zygomatic numbness (V2 sensory distribution) |
| Neck dissection | Not performed |

## Skill Decisions Applied

### 1. Neural Pathway Coverage → TIER 1 (NAMED NERVE V2)
- **Evidence**: MRI widening + clinical facial numbness = confirmed V2 involvement
- **Rule**: "Pathology/imaging explicitly names an involved nerve → Cover ENTIRE ipsilateral CAVERNOUS SINUS to Meckel's cave level"
- **Pathway**: Primary → greater palatine nerve → pterygopalatine fossa → foramen rotundum → cavernous sinus lateral wall → Meckel's cave
- **Contouring**: 5-8mm in soft tissue, 3-5mm within bony canals

### 2. Neck Nodes → DO NOT IRRADIATE
- Primary site = hard palate → ABOVE oral fissure
- Histology = cribriform-tubular → non-solid type
- **Rule** (Section 2.3): "Above oral fissure AND non-solid histology → NO prophylactic neck RT"

### 3. Tumor Bed
- Full-thickness hard palate defect → include nasal floor mucosa
- Posterior extension to soft palate junction
- Ipsilateral inferior nasal meatus involvement

## Final Prescription

| PTV | Dose | Coverage |
|-----|------|----------|
| PTV-60 | 60 Gy / 30 fx | Tumor bed + V2 pathway (PPF → foramen rotundum → entire ipsilateral cavernous sinus) |
| Neck | 0 Gy | No prophylactic neck irradiation |

## Key OAR Constraints

| OAR | Limit | Reason |
|-----|-------|--------|
| Optic chiasm | Dmax ≤ 54 Gy | Adjacent to cavernous sinus |
| Brainstem | Dmax ≤ 54 Gy | Posterior to cavernous sinus |
| Ipsilateral temporal lobe | Dmax ≤ 60 Gy | V2 pathway runs near sphenoid |
| Ipsilateral lacrimal gland | Dmean ≤ 30 Gy | Near V2 infraorbital pathway |
| Ipsilateral cochlea | Dmean ≤ 45 Gy | Petrous apex near foramen rotundum |

## One-Line Summary
> Right hard palate ACC (cribriform-tubular), V2 involved → TIER 1. Plan: tumor bed + V2 pathway (PPF → foramen rotundum → entire ipsilateral cavernous sinus) = 60 Gy/30 fx. No prophylactic neck RT (above oral fissure + non-solid type).
